API 端點 方法 功能說明 使用場景
/api/journeys GET 查詢所有旅程 旅程清單展示

🎯 Workshop 挑戰：2 個核心分析模組

模組 1：熱門路線分析 📊

功能說明：統計最多人搭乘的起迄站組合 Top 10

  • ✅ 統計各起迄站組合的搭乘次數
  • ✅ 計算各路線總收入
  • ✅ 依搭乘次數排序

模組 2：尖峰時段分析 ⏰

功能說明：統計各時段（0-23）進站人數分布

  • ✅ 回傳 24 小時完整分布
  • ✅ 識別人數最多時段
  • ✅ 支援班次與人力規劃
